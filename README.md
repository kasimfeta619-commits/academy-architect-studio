# Academy Architect Studio

Уебсайт, създаден за представяне на архитектурни услуги и професионално портфолио.

## Технологии
* HTML
* CSS
* JavaScript

## Как да отворя проекта?
1. Изтеглете файловете от репозиторията.
2. Отворете `index.html` във вашия браузър (Chrome, Firefox, Edge и т.н.).
